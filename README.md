# IQR
The code for our paper ``Invariant quantile regression for heterogeneous environments''.

Explanation of files:

1. The "IQR_demo.py" is the main Python code to reproduce simulations of Model 1 in Section 5, while "iqr_demo_vis1_loss.py", "iqr_demo_vis2_set.py", and "iqr_demo_vis3.py" reproduce Figure 1 in Section 5.
2. The "IQR_demo2.py" is the main Python code to reproduce simulations in Section B.1 of the Supplementary Materials, while "iqr_demo2_h_vis1.py" and "iqr_demo2_h_vis2.py" reproduce Figure 1 in Section B.1 of the Supplementary Materials.
3. The "IQR_demo3.py" is the main Python code to reproduce simulations of Model 2 in Section 5, along with "iqr_demo3_vis_set.py" to reproduce Figure 3 in Section 5.
4. The "IQR_demo5.py" is the main Python code to reproduce simulations of Model 3 in Section 5, along with "iqr_demo5_vis_set.py" to reproduce Figure 4 in Section 5.
5. The "IQR_demo7.py" and "iqr_demo7_vis.py" reproduce Figure 2 in Section 5.
6. Run "gumbel_test.py" to reproduce the Gumbel approximation results in Section B.2 of the Supplementary Materials.
7. Run "real_data_analysis.py" to reproduce Table 1 for real data analysis in Section B.3 of the Supplementary Materials and one can conclude the Table 1 in Section 5.
